# RSS Aggregator

RSSAggregator is a project which was done during passing the [Golang Course](https://www.youtube.com/watch?v=un6ZyFkqFKo&t=31175s) on freeCodeCamp.org


## Installation


```bash
go mod download
```

## Run

```bash
go build && ./rss_aggregator 
```
